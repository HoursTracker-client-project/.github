# HoursTracker Software — Time Tracking Tied to Clients and Projects for Hourly Workers

## About HoursTracker Software

HoursTracker Software provides a complete time tracking environment for hourly workers, freelancers, consultants, contractors, and remote employees who need to log billable hours by client, project, task, and location. HoursTracker Software features start/stop timer, manual entry, GPS location tagging, and break deduction. HoursTracker Software exports timesheets to PDF, CSV, QBO, XLS, and Excel formats. HoursTracker Software offers customizable hourly rates per client, rounding rules, overtime calculation, and built-in reporting. HoursTracker Software includes widgets, notifications, cloud backup, and work week scheduler. HoursTracker Software is trusted by hourly employees, freelance professionals, legal consultants, home care workers, and remote teams worldwide.
